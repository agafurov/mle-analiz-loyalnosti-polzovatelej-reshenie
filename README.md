# mle-analiz-loyalnosti-polzovatelej-reshenie
Авторское решение итогового проекта модуля 2 спринта 7 курса Инженер Машинного Обучения Яндекс Практикума

## Анализ лояльности пользователей Яндекс Афиши

Данный проект представляет собой полноценный аналитический пайплайн. В рамках работы был проведен рефакторинг: исследовательский анализ данных (EDA) перенесен из формата Jupyter Notebook в структурированный Python-код с использованием принципов ООП.

## Описание задачи

Основная цель проекта — загрузка сырых данных из базы PostgreSQL, их очистка, формирование профилей пользователей и проведение исследовательского анализа для выявления факторов, влияющих на возвращаемость клиентов (Retention) и количество заказов.

## Как запустить проект

1. Клонируйте репозиторий и перейдите в папку
2. Создайте и активируйте виртуальное окружение
3. Установите зависимости
4. Настройте доступы к базе данных
5. Положите файл с курсами валют в папку `data/`
6. Запустите пайплайн
